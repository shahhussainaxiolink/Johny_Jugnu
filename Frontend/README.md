# Johnny & Jugnu

A modern, responsive website for Johnny & Jugnu, a premium burger restaurant located in Johar Town, Lahore. Built with React and featuring a sleek dark theme with neon accents, smooth animations, and an intuitive ordering experience.

## 🍔 About

Johnny & Jugnu serves premium smash burgers, loaded fries, and signature wraps with a focus on quality ingredients and bold flavors. Our website showcases our menu, current deals, and provides an easy way for customers to explore our offerings and place orders.

## ✨ Features

- **Interactive Menu**: Browse our complete menu with categories for Burgers, Fries, Wraps, and Drinks
- **Search Functionality**: Quickly find your favorite items with the built-in search feature
- **Shopping Cart**: Add items to your cart and view your order total
- **Flash Deals**: Special combo deals with countdown timers
- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Smooth Animations**: Powered by Framer Motion for a premium user experience
- **Contact Form**: Easy way to get in touch with us
- **Location & Hours**: Clear information about our location and operating hours

## 🛠️ Tech Stack

- **Frontend Framework**: React 19
- **Build Tool**: Vite
- **Styling**: Bootstrap 5 + Tailwind CSS 4
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Linting**: ESLint
- **Package Manager**: npm

## 🚀 Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/johnny-jugnu.git
   cd johnny-jugnu
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Open your browser** and navigate to `http://localhost:5173`

## 📜 Available Scripts

- `npm run dev` - Start the development server
- `npm run build` - Build the project for production
- `npm run preview` - Preview the production build locally
- `npm run lint` - Run ESLint to check for code issues
